# .context

Internal knowledge base for XDS — consumed by AI and humans.

## Structure

```
.context/
├── decisions/      # Architecture Decision Records (ADRs)
├── explorations/   # Research, analysis, brainstorms
├── proposals/      # Feature proposals, RFCs
├── architecture.md # System overview (create when ready)
└── glossary.md     # Terms and conventions (create when ready)
```

## Purpose

- Share state between people and AI instances
- Document architecture decisions and rationale
- Capture explorations and brainstorms
- Not intended as polished public documentation
